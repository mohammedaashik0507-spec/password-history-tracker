## 🆘 TROUBLESHOOTING QUICK START

### "Can't find Node.js"
→ Install from https://nodejs.org/ (v18+)

### "Port already in use"
→ Edit `backend/.env` and change PORT to 5001

### "MongoDB connection failed"
→ Ensure MongoDB running, update MONGODB_URI in backend/.env

### "CORS error"
→ Check CLIENT_URL in backend/.env matches frontend URL

### "npm install fails"
→ Run `npm cache clean --force` then try again

→ See **QUICK_START.md** for more troubleshooting
